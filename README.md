# Brave Anime Toolkit

This project provides a collection of scripts and configurations to enhance the anime viewing experience within the Brave browser.

## Files

- `anime-brave-fix.user.js`: A Userscript to fix common issues or add features to anime streaming sites.
- `checksums.sha256`: SHA256 checksums for verifying the integrity of the project files.
- `install_guide.md`: Markdown file containing instructions on how to install and set up the toolkit.
- `install.ps1`: PowerShell script for automated installation.
- `LICENSE`: The license under which this project is distributed.
- `manifest.json`: Manifest file for browser extensions or related configurations.
- `profile_prefs.json`: Preferences file for Brave browser profiles.

## Installation

Refer to `install_guide.md` for detailed installation instructions.

## Usage

Once installed, the `anime-brave-fix.user.js` script will automatically apply its fixes/features to the supported anime streaming websites when using the Brave browser.
